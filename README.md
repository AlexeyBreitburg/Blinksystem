All scripts are oriented to the architecture of ARM, or to be more precise for the Raspberry Pi / Orange Pi microcomputer. To start, recommend the user name **blink**, and use at least a minimal Debian assembly with full kernel functionality, also requires Python version 3.4 or higher. You must install libraries that are used in scripts. You can install all the libraries:
```
sudo pip3 install gtts pygame colorama wget
```
And remove installed libraries you can using this command:
```
sudo pip3 remove gtts pygame colorama wget
```
